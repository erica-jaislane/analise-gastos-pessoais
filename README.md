# 💰 Análise de Gastos Pessoais

Projeto desenvolvido na disciplina de **Análise de Dados com Python** — 5º semestre de Engenharia de Software.

## 📌 Problema

Entender para onde vai o dinheiro ao longo dos meses: quais categorias consomem mais, qual forma de pagamento é mais usada e prever o total de gastos do mês seguinte.

## 🛠️ Tecnologias utilizadas

| Biblioteca | Uso |
|---|---|
| `Pandas` | Limpeza, transformação e agrupamento dos dados |
| `NumPy` | Operações numéricas e criação de features |
| `Matplotlib` | Geração de gráficos e visualizações |
| `Scikit-learn` | Modelo de Regressão Linear para previsão |
| `Streamlit` | Interface web interativa |
